# Changelog

## v1.0 - First release

- Market tab: full scanned shop price list, searchable, filterable, sortable.
- Conversions tool: convert any item between Item / Stack / Shulker / Double Chest / Inventory units, with live value calculation.
- Buy Command tool: generates ready-to-paste `/wtb buy <item> <amount> <price>` commands.
- Shopping List tool: build a list with quantity presets, automatic market/crafted pricing, recursive crafting-cost breakdown, and editable rows.
- Item icons pulled directly from game textures (confirmed-good + needs-review sets).
- Self-contained offline version available as a downloadable zip.
